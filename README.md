QuickChat - Real-Time Chat Application

A full-stack real-time chat application built using the MERN stack and Socket.IO. QuickChat enables users to communicate instantly through a modern and responsive interface with real-time messaging capabilities.

📖 About The Project

QuickChat is a real-time messaging platform developed using MongoDB, Express.js, React.js, Node.js, and Socket.IO. The application allows users to create accounts, log in securely, and exchange messages instantly.

The project is deployed on Vercel and uses MongoDB Atlas for cloud database storage.

⚙️ Installation
Clone Repository
git clone https://github.com/venisha28/QuickChat-Full-Stack.git
Install Frontend Dependencies
cd client
npm install
npm run dev
Install Backend Dependencies
cd server
npm install
npm start
🔐 Environment Variables
Client (.env)
VITE_BACKEND_URL=YOUR_BACKEND_URL
Server (.env)
PORT=5000
JWT_SECRET=YOUR_SECRET_KEY
MONGODB_URI=YOUR_MONGODB_CONNECTION_STRING
CLOUDINARY_CLOUD_NAME=YOUR_CLOUD_NAME
CLOUDINARY_API_KEY=YOUR_API_KEY
CLOUDINARY_API_SECRET=YOUR_API_SECRET
